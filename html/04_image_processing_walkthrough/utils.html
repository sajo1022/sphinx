  • xyzi_image (list) – 4-D numpy array containing x, y, z and intensity values.

  • save_path (str) – Path images are saved to.

image_processing.ip_utils.pointcloud_to_ros_style_pointcloud(pointcloud, include_intensity=False, intensity_values=None)

Creates a request for the image processinge stone location services (fitting stones and standard stones)

Parameters

  • pointcloud (list) – [[float, float, float], …] Pointcloud as a list of x,y,z coordinates.

  • include_intensity (bool) – True/False to state if intensity information is included.

  • intensity_values (float) – List of float values that correspond to the intensity of a pointcloud coordinate.

Returns

ros_pc

Return type

Pointcloud (and intesity) as LocateStandardStoneSrv (LocateFittingStoneSrv).
